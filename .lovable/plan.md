

# דחיסת כל תמונות האתר לפורמט WebP

## מה ייעשה
סקריפט אוטומטי ידחוס את כל התמונות בפרויקט (JPG/PNG) לפורמט WebP קל יותר.

## תיקיות יעד
- `src/assets/` — כולל `hero-image.jpg` (6.3MB) ושאר נכסים (~11MB סה"כ)
- `public/lovable-uploads/` — 121 קבצים (~48MB סה"כ)

## הגדרות דחיסה
- פורמט: **WebP**
- איכות: 80%
- רזולוציה מקסימלית: 1400px רוחב
- קבצים מתחת ל-50KB לא יידחסו

## שלבים
1. הרצת סקריפט Python עם Pillow להמרת כל JPG/PNG ל-WebP
2. עדכון כל ההפניות בקוד (imports, src attributes) מ-`.jpg`/`.png` ל-`.webp`
3. מחיקת הקבצים המקוריים

## חיסכון צפוי
מ-~59MB ל-~8-12MB (חיסכון של ~80%)

## פרטים טכניים
- שימוש ב-Pillow (`pip install Pillow`) להמרה
- סריקת כל קבצי TSX/TS/CSS לעדכון נתיבי תמונות
- תמונות SVG לא יושפעו
